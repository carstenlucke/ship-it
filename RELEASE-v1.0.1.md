# Release v1.0.1 — Ship It!

Wartungsrelease mit verbesserter Release-Infrastruktur und aktualisierter Dokumentation.

## Infrastruktur

- **Eigener Release Letter im GitHub Release**: Der Release-Workflow verwendet ab sofort den projektspezifischen Release Letter (`RELEASE-v<version>.md`) anstelle der automatisch generierten GitHub Release Notes. Das Release auf GitHub enthält damit eine kuratierte, deutschsprachige Beschreibung der Änderungen.
- Aktualisierte Projektdokumentation (README und CLAUDE.md)
- Obsolete Spezifikationsdateien aus dem Repository entfernt
