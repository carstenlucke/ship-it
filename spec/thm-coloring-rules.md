# THM Farbgebung – Auszug CD-Manual

## 1. Hauptfarben

Die **Hauptfarben** Grün und Grau prägen das Erscheinungsbild der THM.  
Sie sind in Pantone, CMYK, RAL sowie für den Bildschirmeinsatz in RGB und Hexadezimalwerten definiert.  
Im Fünf-Farben-Druck wird Grün (Pantone 368) als Sonderfarbe eingesetzt.

### 1.1 THM Grün

- Bezeichnung: THM Grün  
- Pantone: 368  
- CMYK (ISO coated v2): 57 / 0 / 100 / 0  
- CMYK TZ (ISO newspaper 26v4): 36 / 0 / 95 / 2  
- RGB (sRGB): 128 / 186 / 36  
- Hex: #80BA24  
- RAL: 6018 (Gelbgrün)

### 1.2 THM Grau

- Bezeichnung: THM Grau  
- Pantone: 7546  
- CMYK (ISO coated v2): 33 / 4 / 0 / 72  
- CMYK TZ (ISO newspaper 26v4): 33 / 11 / 5 / 48  
- RGB (sRGB): 74 / 92 / 102  
- Hex: #4A5C66  
- RAL: 7015 (Schiefergrau)
